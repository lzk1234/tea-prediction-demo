# 🍵 高山茶销量预测系统

基于深度学习(LSTM)的销量预测模型可视化平台

## 使用说明

这是一个使用 Streamlit 构建的高山茶销量预测演示应用，包含：
- 数据集可视化分析
- LSTM 深度学习销量预测
- 预测对比分析

## 技术栈

- Streamlit - Web 应用框架
- Pandas - 数据处理
- Plotly - 数据可视化
- NumPy - 数值计算
